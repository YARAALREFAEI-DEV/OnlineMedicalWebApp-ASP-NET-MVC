# 🏥 Online Medical System

This is a web-based application developed using **ASP.NET MVC** and **SQL Server** . 
The system allows patients, doctors, and administrators to manage Medical system operations efficiently through a modern web interface.


#  Features

- Doctor, Patient, and Admin roles with secure login
-    Role-based login system (Admin, Doctor, Patient)
- Online Appointment booking system
- Doctor profile management 
- Admin dashboard for managing doctors, patients, and appointments
- Responsive UI using Bootstrap
- Data stored in SQL Server Database

# 🛠 Technologies Used

- ASP.NET MVC (.NET Framework)
- Entity Framework
- C#
- SQL Server
- Bootstrap
